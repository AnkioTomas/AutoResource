::: note 分类
自动记账和主流记账软件一样支持二级分类。
:::

## 一级分类

从首页进入`分类管理`，点开右下角`+`号，会出现两个选项：`切换账本`和`添加分类`。

确保正在使用的账本是你要操作的账本（图中橙色方框），然后点击`添加分类`按钮`。

:::warning
每个账本的分类是不共享的，需要**每个账本单独设置**。
:::

在输入框中填入分类名，即可完成。**注意：** 分类名必须与记账App的一致。

## 二级分类
点击相应分类，展开列表后，点击添加子类，输入分类名称后确定即可。
