记账应用配置

自动记账是一款辅助记账的插件，其账单记录功能和账单查询功能还需要其他的App来配合。 目前已支持自动记账的App有:

点击折叠/查看 open

钱迹 (opens new window) (官方推荐)

# 钱迹配置教程（非常重要）

# 第一步、下载并安装：钱迹 (opens new window)

# 第二步、登录钱迹

提示

打开钱迹，注册然后登陆，如图所示：（这个应该不需要手把手教了吧？毕竟都是读过书的人。）

# 第三步、设置资产

注意

如果你不想区分记录账单的账户可以不用设置资产，在自动记账中打开懒人模式后，直接进入第四步、设置分类

提示

登录后，点开右上角抽屉，进入设置，见下图：

提示

在设置页面，进入实验室，找到资产管理，打开开关，然后回到主页面，如图所示：

提示

回到主页后，向左滑动或点击右上角钱包图案，进入资产页面。然后点击右下角，添加资产。如图：

# 第四步、设置分类

钱迹默认有一些分类，如果你对此并不满意，那么，我们就需要修改它。如果你不需要分类，直接进入自动记账权限配置

提示

回到主页，点击下方的+号

提示

点击管理，修改分类，这里如何修改我就不再赘述了

注意

我们还有一件事情要做：允许钱迹自启动，这很重要！！！！ 否则你将进入随缘记账时代。

钱迹其他功能，大家可以自行查看钱迹的官方文档 (opens new window)。
